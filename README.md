# DevOps Capstone Project

This package contains a complete DevOps capstonee project scaffold for the repository:

- CI/CD workflows
- Dockerfiles and `.dockerignore`
- Docker Compose for local development
- Kubernetes manifests
- Secret management examples
- Monitoring setup docs
- Grafana dashboard JSON files

## Important placeholders to update
- Replace `YOUR_DOCKERHUB_USERNAME` inside Kubernetes deployment manifests
- Replace `finch.example.com` with your actual domain
- Replace placeholder secret values before real deployment

## Suggested submission flow
1. Review all YAML and markdown files
2. Update image names and domain
3. Push the files into your repository
4. Take screenshots of GitHub Actions, Docker images, and Kubernetes resources for your final report
